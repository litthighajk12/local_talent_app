# 🎯 Local Talent Showcase App

## 📌 Overview

Local Talent Showcase is a Flutter-based mobile and web application that helps users discover and register for nearby events based on their interests and location.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* Talent-based profile creation
* Search events (normal + location-based)
* Event registration
* Profile management
* Notifications (event updates)

### 📍 Smart Recommendation

* Shows events based on:

  * User location (nearby search)
  * User talents (future enhancement)

### 🔔 Notifications

* In-app notifications for:

  * New events
  * Event updates

### 🛠 Admin Features (Planned)

* Add / Edit / Delete events
* View registrations

## ▶️ How to Run

### 1. Clone Project

```bash
git clone <your-repo-link>
cd local_talent_app
```

### 2. Install Dependencies

```bash
flutter pub get
```

### 3. Run App

```bash
flutter run
```

---

## 📱 Run on Emulator

* Open Android Studio → Device Manager
* Start emulator
* Run `flutter run`

---

## 📲 Run on Real Device

* Enable Developer Mode
* Enable USB Debugging
* Connect phone

```bash
flutter run
```

---

## 🌐 Run on Web

```bash
flutter run -d chrome
```
